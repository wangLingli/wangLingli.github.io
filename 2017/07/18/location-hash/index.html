location.hash

Veröffentlicht am 2017-07-18

最近看了一个用swiper.js做的活动页项目，当滑到第六页点击“了解详情”按钮，可以使用指定相应的url跳转到详情页，那么当点击详情页中“返回”按钮，怎么返回到swiper第六页的活动页面呢？



window.location.href

location是javascript里边管理地址栏的内置对象,window.location.href可以获取到完整的url，比如window.location.href=”www.babytree.com”表示的是重新定向，页面跳转到新的页面。

window.location.hash

“#”代表网页中的一个位置,其右面的字符，就是该位置的标识符。比如，

http://www.babytree.com#tool

就代表网页的tool位置。浏览器读取这个URL后，会自动将tool位置滚动至可视区域。

为网页位置指定标识符，有两个方法。一是使用锚点，比如<a name="tool"></a>，二是使用id属性，比如<div id="tool" >。

获取的是页面的锚链接，相比如href,通过window.location.hash并不会跳转到新的链接，只会在当前链接里面改变锚链。例如<a href="www.babytree.com#tool">通过window.location.hash得不到完整的链接（URL），仅得到 #tool.

window.location.hash这个属性可读可写。读取时，可以用来判断网页状态是否改变；写入时，则会在不重载网页的前提下，创造一条访问历史记录。

每一次改变#后的部分，都会在浏览器的访问历史中增加一个记录，使用”后退”按钮，就可以回到上一个位置。这对于ajax应用程序特别有用，可以用不同的#值，表示不同的访问状态，然后向用户给出可以访问某个状态的链接。

值得注意的是，上述规则对IE 6和IE 7不成立，它们不会因为#的改变而增加历史记录。

var hash = window.location.hash;  // #referno5
