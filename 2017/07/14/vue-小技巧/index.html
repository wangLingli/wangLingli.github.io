防止花括号闪烁

打开页面有时候会出现 {{msg}} 这种表达式的情况，这是因为 JavaScript 会等DOM加载完成之后，才会去操作相应的DOM元素进行数据渲染。

使用v-text

在vue中可以使用 v-text 指令渲染数据，这样可以避免 {{ }} 的产生，如果里面包含需要解析的 html 元素，则使用 v-html即可。

使用v-cloak

v-cloak一般用在大段落中包含大量 {{ }} 的div上，这个指令保持在元素上直到关联实例结束编译。和 CSS 规则如 [v-cloak] { display: none } 一起用时，这个指令可以隐藏未编译的 Mustache 标签直到实例准备完毕。

slot插槽

slot是一个很有用的特性，可以用来向组件内部插入一些内容。slot就是“插槽”的意思，相当先定义好位置，由用户来决定插入的内容，这样就不会直接覆盖原有的内容。

默认情况下页面只会显示“我是子组件”，而ul里的内容不显示：

匿名slot

单个slot使用最简单，也是最常用的，父组件在使用的这个组件的时候，想在内部自定义一些初始化数据，这时候就可以用slot实现。

当child组件内有多个标签也会一起插入到slot的位置。

具名slot

  • 将放在子组件里的不同html标签放在不同的位置
  • 父组件在要分发的标签里添加 slot=”name名” 属性
  • 子组件在对应分发的位置的slot标签里，添加name=”name名” 属性，
  • 然后就会将对应的标签放在对应的位置了。
